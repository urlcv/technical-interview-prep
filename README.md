# Technical Interview Prep

Technical interview preparation tool with algorithmic pattern training, complexity drills, guided problem-solving, and spaced repetition.

**Live tool:** [urlcv.com/tools/technical-interview-prep](https://urlcv.com/tools/technical-interview-prep)

## Features

- **25+ algorithm problems** across arrays, strings, trees, graphs, DP, and more
- **Guided problem-solving flow** — 11-step structured interview simulation
- **Pattern recognition drills** — identify which algorithmic pattern applies
- **Complexity drills** — analyse code snippets for time and space complexity
- **Mock interview mode** — timed simulations with follow-up questions
- **Flashcards & rapid revision** — spaced-repetition concept cards
- **Focus Map dashboard** — see strengths, weaknesses, and a clear next action
- **Spaced repetition (SM-2)** — weak problems resurface at optimal intervals

### Designed to keep you moving

- Session modes by available time (10 / 30 / 60 min)
- Warm-up rounds to build momentum
- Sentence starters and templates to get past blank-page moments
- Gentle nudges when you're stuck
- Quick-exit bookmarking with instant resume
- Streak system to build consistency
- Keyboard shortcuts for every action

## Installation

```bash
composer require urlcv/technical-interview-prep
```

The package auto-discovers its service provider via Laravel's package discovery.

## How it works

Everything runs entirely in the browser via Alpine.js. No server calls, no accounts needed. Progress is saved to `localStorage`.
